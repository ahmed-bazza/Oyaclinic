# OYA Hermitage Medical Clinic Website

## Files
- `index.html` — one-page clinic website
- `CNAME` — custom domain for GitHub Pages: oyaclinic.ca

## GitHub Pages setup
1. Create a GitHub repository, for example `oyaclinic.ca`.
2. Upload `index.html` and `CNAME` to the repository root.
3. Go to **Settings > Pages**.
4. Set the publishing source to the `main` branch and root folder.
5. In **Custom domain**, enter: `oyaclinic.ca`.
6. Configure DNS at your domain provider according to GitHub Pages instructions.
7. Enable HTTPS once GitHub completes the DNS check.

## Editing
Open `index.html` and change any text, phone, fax, hours, or booking URL directly.
